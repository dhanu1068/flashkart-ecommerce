# FlashKart Ecommerce REST API

## Overview
FlashKart is a simple Java Spring Boot backend application that exposes RESTful APIs
to manage ecommerce products using **in-memory storage**.

The application is designed as a lightweight backend service to demonstrate:
- REST API design
- Input validation
- Constructor-based dependency injection
- In-memory data handling without a database

---

## Technology Stack
- Java 17
- Spring Boot
- Spring Web
- Bean Validation (Jakarta Validation)
- Maven
- In-memory storage (ArrayList)

---

## How to Run the Application

### Prerequisites
- Java 17 or higher
- Maven
- IntelliJ IDEA (recommended)

### Steps
1. Clone the repository:
2. Open the project in IntelliJ IDEA.

3. Run the application using one of the following methods:

    - From IntelliJ:
      Run the `FlashkartApplication` main class.

    - From terminal:
      ```bash
      mvn spring-boot:run
      ```

4. Once started, the application will be available at:
   ```text
   http://localhost:8080
